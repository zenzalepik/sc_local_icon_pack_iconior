Cara Menggunakan Font Awesome CDN

Untuk menggunakan ikon dari Font Awesome CDN, tambahkan kode berikut di bagian <head> dokumen HTML Anda:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

Kemudian gunakan kode ikon seperti contoh berikut:

<i class="fas fa-home"></i> 
<i class="fas fa-user"></i> 
<i class="fas fa-star"></i>

Untuk ikon solid gunakan fas, untuk regular gunakan far, untuk brands gunakan fab.

Kode ikon berhasil disalin ke clipboard!
