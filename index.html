Cara Menggunakan Iconior CDN

Untuk menggunakan ikon dari Iconior CDN, tambahkan kode berikut di bagian <head> dokumen HTML Anda:

<link rel="stylesheet" href="https://cdn.iconior.com/icons/all.css">

Kemudian gunakan kode ikon seperti contoh berikut:

<i class="ic-home"></i> 
<i class="ic-user"></i> 
<i class="ic-star"></i>

Untuk ukuran dan warna yang berbeda, Anda dapat menambahkan kelas CSS atau menggunakan gaya inline.

Kode ikon berhasil disalin ke clipboard!
