Cara Menggunakan Iconoir

Untuk menggunakan ikon dari Iconoir, tambahkan CDN berikut di bagian <head> dokumen HTML Anda:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/iconoir-icons/iconoir@main/css/iconoir.css">

Kemudian gunakan kode ikon dengan format iconoir-[nama-ikon]:

<i class="iconoir-home"></i>
<i class="iconoir-user"></i>
<i class="iconoir-search"></i>

Bonus: Anda bisa mengubah ukuran dengan CSS:

.iconoir-home {
    font-size: 32px;
    color: #667eea;
}
Kode ikon berhasil disalin!
